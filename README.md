# Excel Import & Merge (WPF)

A WPF desktop application built using C# and .NET Framework that allows users to:
- Select multiple Excel (XLS/XLSX) files
- Read data from each file
- Merge all data into a single table
- Display merged data in a grid

## Technologies
- C#
- .NET Framework
- WPF
- EPPlus

## Features
- Multi-file selection
- Excel import
- Data merge
- Simple UI

## How to Run
1. Open the solution in Visual Studio
2. Restore NuGet packages
3. Build and run the project
